## 来源类型

| 类型 | 示例 |
| --- | --- |
| 活动资料 | 活动手册、流程表、签到记录 |
| 影像资料 | 照片批次、授权截图、精选图册 |
| 口述资料 | 访谈录音、转写稿、校对稿 |
| 工艺文档 | 面具维护、服装收纳、运输规范 |
| 数字记录 | 资料馆发布记录、索引变更、版本记录 |
