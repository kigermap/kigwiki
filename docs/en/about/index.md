<section class="archive-page-intro" markdown>
  <p class="archive-kicker">About</p>
  <h1>About</h1>
  <p>Kigurumi Chronicle is a static archive for the kigurumi community. It prioritizes public introduction, long-term collaboration, permission notes, and room for future content growth instead of adding too many sections during early development.</p>
</section>

<section class="archive-section" markdown>
  <p class="archive-kicker">Purpose</p>
  <h2>Project Goals</h2>
  <p>The site turns public community information into a stable web presence: clear entry points, clear copyright boundaries, room for multilingual growth, and low-cost maintenance through GitHub and Cloudflare Pages.</p>

  <div class="archive-feature-grid archive-feature-grid--compact">
    <article class="archive-feature">

    <article class="archive-feature">
