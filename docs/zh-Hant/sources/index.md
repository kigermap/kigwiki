  <h1>來源目錄</h1>
  <p>來源目錄是編年志可信度的根基。沒有來源，紀事只能算線索；有了來源、授權和版本說明，資料纔可以被公開引用、複覈和繼續修訂。</p>
</section>

  <div markdown>
    <p class="archive-kicker">Citation Discipline</p>
    <h2>引用規範</h2>
    <p>每條來源建議記錄標題、作者或提供者、日期、原始位置、可訪問狀態、授權說明、引用格式和備註。若來源來自私人整理，應先確認是否可以公開署名或只可內部保存。</p>
  </div>
  <ol class="archive-steps">
    <li><strong>確認來源</strong><span>知道資料來自哪裏，誰可以解釋它</span></li>
    <li><strong>確認授權</strong><span>知道是否能公開、是否需署名、是否需打碼</span></li>
    <li><strong>確認版本</strong><span>記錄發佈日期、修訂時間和可能的版本差異</span></li>
    <li><strong>綁定條目</strong><span>把來源連接到對應的年份、地點、人物和事件</span></li>
  </ol>
</section>
