<section class="archive-page-intro" markdown>
  <p class="archive-kicker">About</p>
  <h1>關於介紹</h1>
  <p>Kigurumi 編年志是一個面向 kigurumi 社群的靜態資料館。它優先服務公開介紹、長期協作、資料授權說明和後續內容擴展，而不是在開發早期堆出過多欄目。</p>
</section>

<section class="archive-section" markdown>
  <p class="archive-kicker">Purpose</p>
  <h2>項目目標</h2>
  <p>這個站點希望把社群相關的公開資訊整理成更穩定的網頁形態：有清晰入口、有版權邊界、有多語言擴展空間，也能透過 GitHub 和 Cloudflare Pages 保持低成本維護。</p>

  <div class="archive-feature-grid archive-feature-grid--compact">

      <p>頁面結構克制，優先呈現讀者最需要了解的項目定位和參與方式。</p>
    </article>
    <article class="archive-feature">

    </article>
    <article class="archive-feature">
      <span class="archive-feature__icon">護</span>

      <p>無資料庫、無後端，使用靜態構建和自動部署降低長期維護成本。</p>
    </article>
  </div>
